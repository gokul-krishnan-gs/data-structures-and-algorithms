- Step-by-step algorithm:

### Code
```js

let biggest = numbers[0];

for (let i = 1; i < numbers.length; i++) {
  if (numbers[i] > biggest) {
    biggest = numbers[i];
  }
}

console.log("Biggest number is:", biggest);
```
